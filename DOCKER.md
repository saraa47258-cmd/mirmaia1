# تشغيل وعرض التطبيق من Docker

## 1. تشغيل كل الخدمات

من مجلد المشروع (حيث يوجد `docker-compose.yml`):

```powershell
cd c:\Users\HP\Desktop\mirmaia1-master
docker-compose up -d --build
```

انتظر حتى يكتمل البناء والتشغيل (أول مرة قد تستغرق عدة دقائق).

---

## 2. عناوين العرض (من المتصفح)

بعد التشغيل افتح المتصفح على:

| الخدمة | الرابط | الوصف |
|--------|--------|--------|
| **الواجهة (التطبيق)** | **http://localhost:3001** | نظام نقاط البيع — الدخول والكاشير والمنيو |
| **منيو الزبائن** | http://localhost:3001/menu/view | عرض المنيو للزبائن فقط |
| **باركود المنيو** | http://localhost:3001/menu/barcode | صفحة رمز QR لفتح المنيو |
| **الـ API (الخادم)** | http://localhost:3000 | للاستخدام الداخلي |
| **phpMyAdmin** | http://localhost:8081 | إدارة قاعدة البيانات |

---

## 3. الدخول للتطبيق

1. افتح **http://localhost:3001**
2. سجّل الدخول (مثلاً: `admin@mirmaia.com` وكلمة المرور)
3. من القائمة الجانبية يمكنك فتح: الكاشير، منيو الزبائن، باركود المنيو، لوحة التحكم، وغيرها.

---

## 4. إيقاف الخدمات

```powershell
docker-compose down
```

---

## 5. عرض من جهاز آخر على الشبكة

إذا أردت فتح التطبيق من جوال أو جهاز آخر على نفس الشبكة:

1. اعرف IP جهازك (مثلاً من `ipconfig` على Windows).
2. على الجهاز الآخر افتح: **http://IP_جهازك:3001**  
   مثال: `http://192.168.1.10:3001`
3. إن لم يعمل الـ API (مثل الصور أو تسجيل الدخول)، قد تحتاج لإعادة بناء الـ frontend مع عنوان الـ API الصحيح (مثلاً `http://IP_جهازك:3000`) ثم إعادة `docker-compose up -d --build frontend`.

---

## 6. التحقق من أن الخدمات تعمل

```powershell
docker-compose ps
```

يفترض أن ترى الحاويات (frontend, backend, mysql, phpmyadmin) بحالة **Up**.
